# What is this?
Probably nothing useful to you. This bot was made primarily to make one specific Discord TTRPG slightly easier and more fun.

# Prerequisites
- cargo
- rust (>=2024 edition)

# Run instructions
1. Clone the repo into a new folder with `git clone`.
2. `cd` into the new folder and run with `cargo run`.
3. The first time you run it, you'll need to give it a token to use. It'll then create a config file, which you'll need to change to work for your preferences.
4. Once you've filled out the config file, rerun the bot and you're good to go.
